What is HTTP?

HTTP is short for Hyper Text Transfer Protocol. It is the most convenient way to move data quickly and reliably on the web. If two computers want to communicate they must both speak according to the HTTP protocol.

For example, when you want to use any type of website you need to be connected to the internet to be able to get information from the server. When you type in your webpage in the browser your computer will send a request to the server using an HTTP protocol. The server then responds with another HTTP protocol message back to your local computer so you can access the website you searched for.
